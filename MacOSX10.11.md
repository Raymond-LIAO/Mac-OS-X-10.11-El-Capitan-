## OS X Preferences
```
%将Library文件夹设置为显示
chflags nohidden ~/Library

%将截屏保存到桌面的Screenshots子文件夹
mkdir ~/Desktop/Screenshots
defaults write com.apple.screencapture location ~/Desktop/Screenshots

%方便通过双指滑动 Dock 中应用的图标，来查看这个应用当前开启的窗口
defaults write com.apple.dock scroll-to-open -bool true && \ killall Dock

%Dock 栏只显示已打开的应用程序
defaults write com.apple.dock static-only -boolean true; killall Dock
%修改回来
defaults delete com.apple.dock static-only; killall Dock

%让崩溃报告在「通知中心」显示
defaults write com.apple.CrashReporter UseUNC 1
%修改回来
defaults write com.apple.CrashReporter UseUNC 0
```

## Things to try with Zsh and Oh-my-Zsh
终于忍不住折腾了一下自己的terminal，用上了早就闻名已久的Zsh和[Oh-my-Zsh](https://github.com/robbyrussell/oh-my-zsh)。

### 安装Zsh
Mac上面自带，所以这一步略去
### 安装Oh-my-Zsh
`sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"`
### 个性化配置
来到.zshrc文件中，做你喜欢的修改把，比如：

```
alias cls='clear'
alias ll='ls -l'
alias la='ls -a'
alias vi='vim'
alias javac="javac -J-Dfile.encoding=utf8"
alias grep="grep --color=auto"
alias -s html=subl   # 在命令行直接输入后缀为 html 的文件名，会在 TextMate 中打开
alias -s py=subl       # 在命令行直接输入 python 文件，会用 vim 中打开，以下类似
alias -s js=subl
alias -s c=subl
alias -s java=subl
alias -s txt=
alias -s gz='tar -xzvf'
alias -s tgz='tar -xzvf'
alias -s zip='unzip'
alias -s bz2='tar -xjvf'
```

### 主题
在~/.oh-my-zsh/themes位置，默认是：ZSH_THEME=”robbyrussell”。
### 还有茫茫多的插件
比如我们最常用的Git，还有各种补全，尽在~/.oh-my-zsh/plugins

### Add Sublime Text CLI

	sudo rm /usr/local/bin/subl
	sudo ln -s /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl

当然如果发现提示不存在文件夹时，需要先

	touch /usr/local/bin

##关于如何备份你的电脑
最简单的方式当然是选择一块移动硬盘加上Time Machine，当然还有很多联网啊搭建NAS之类的高大上但是更加方便的方式，抑或者也写一篇像本文这样的文章，详细记录下你的（装机）过程，就都不在这里详述了。
在这里要给大家安利的是来自于homebrew的mackup，[Mackup](https://github.com/lra/mackup)是 Github 上的一个开源项目，旨在帮助你快速方便地备份恢复软件的配置文件，Mackup 已经支持了包括 iTunes Applescripts 在内的一系列常用的软件程序，可以在它的项目首页进行更多了解。
在已经安装了homebrew的基础上，只需要简单一句：
`brew install mackup`
就OK了

默认的储存位置是dropbox，一般应该没有什么问题，也可以自主修改到icloud或者google drive。

#####常用命令

* `mackup backup` 使用 Mackup 进行备份操作
* `mackup restore` 使用 Mackup 进行数据的恢复
* `mackup list` 查看 Mackup 支持的软件列表
* `mackup -h Mackup` 的帮助命令
